Concatenation:

  * Can be used if you want to add 2 strings.
      e.g:
          print("Hello, " + "World!") //Hello, Workd!

  * Can be used if you want to add 2 numbers.
      e.g:
          print(5 + 6)

  * But you can't used if you want to add string and a number.
      e.g:
         print("The sum of 2 plus 2 is " + 4)
