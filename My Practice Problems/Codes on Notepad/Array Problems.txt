Reverse of Array

Code:
#include <stdio.h>

int main()
{  
    int array[5];
    for (int i=0 ;i<5; i++)
    {
        printf("Enter number %d\n",i+1);
        scanf("%d",&array[i]);
    }
    
    for (int i=4; i>=0 ;i--)
    {
        printf("{%d,}",array[i]);
    }
}

OUTPUT:

Enter number 1
4
Enter number 2
6
Enter number 3
2
Enter number 4
7
Enter number 5
9
{9,}{7,}{2,}{6,}{4,}
